# 📩 Spam Detection using Machine Learning

A Machine Learning-based Spam Detection System that classifies text messages as **Spam** or **Not Spam (Ham)** using Natural Language Processing (NLP) and Machine Learning.

---

## 📖 Project Overview

This project analyzes text messages and predicts whether they are **Spam** or **Not Spam (Ham)** using NLP preprocessing and a trained machine learning model.

### Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- NLTK
- Scikit-learn
- Pickle

---

## ✨ Features

- Spam Detection
- Ham Detection
- Real-time Prediction
- Streamlit Web Interface
- NLP-based Text Processing
- Fast and Lightweight
